Sobre Programação

    Durante o curso de ADS eu já desenvolvi algumas soluções utilizando a linguagem C#, como atividades para a obtenção de nota em algumas disciplinas. E então eu coloquei ao lado imagens de uma dessa atividade, ela se chama depreciação de veículos e realiza cálculos com essa finalidade

Página Inicial da Solução
Essa é a tela inicial da solução, nela estão os botões para iniciar as outras funcionalidades.

Funcionalidade Cadastrar Veículo
Essa é a funcionalidade responsável por cadastrar os veículos no banco de dados.

Funcionalidade Lista Simples
Essa é a funcionalidade que gera uma tabela simples com alguns dados dos veículos.

Funcionalidade Lista Geral
Essa funcionalidade gera uma tabela com todos os dados dos veículos cadastrados.

Funcionalidade Consulta Veículo
Essa funcionalidade fornece um campo de busca(por placa) para ver os dados de veículos específicos.

Funcionalidade Lista Seminovos
Essa funcionalidade exibe uma tabela com os carros seminovos(3 ou menos anos de uso).

Funcionalidade Lista Usados
Essa funcionalidade exibe uma tabela com os carros usados(mais de 3 anos de uso).

Funcionalidade Lista +5 anos de uso
Essa funcionalidade exibe uma tabela com os carros usados com mais de 5 anos de uso.
